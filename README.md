# 🎯 Word Match Puzzle

เกมฝึกคำศัพท์ภาษาอังกฤษ-ไทยแบบ Interactive สำหรับเด็กและผู้ที่สนใจ

![PWA](https://img.shields.io/badge/PWA-Enabled-brightgreen)
![License](https://img.shields.io/badge/License-MIT-blue)

## 🌟 ฟีเจอร์

### 🎮 การเล่นเกม
- ✅ **Drag & Drop** - ลากเส้นจับคู่คำศัพท์ภาษาอังกฤษกับคำแปลภาษาไทย
- ✅ **Text-to-Speech** - ฟังเสียงอ่านคำศัพท์ภาษาอังกฤษ (Web Speech API)
- ✅ **เลือกความยาก** - เลือกจำนวนคำศัพท์ได้ (10/15/20/30 คำ)
- ✅ **ระบบคะแนน** - 10 คะแนนต่อคำตอบที่ถูกต้อง
- ✅ **จับเวลา** - ติดตามเวลาที่ใช้ในการเล่นเกม
- ✅ **High Score** - บันทึกคะแนนสูงสุด

### 📊 การติดตามผล
- 📅 แสดงวันที่และเวลาที่เล่น
- ✅/❌ แสดงรายการคำศัพท์ที่ตอบถูก/ผิด
- 📋 สรุปผลการเรียนรู้

### 📱 Progressive Web App (PWA)
- 📲 ติดตั้งบนมือถือ/แท็บเล็ตได้
- 🌐 เล่นแบบ Offline ได้ (ไม่ต้องใช้อินเทอร์เน็ต)
- ⚡ โหลดเร็ว ทำงานได้ทันที
